主要功能

📍 GPS 里程

watchPosition + Haversine 計距，含訊號濾波；訊號差時退回時間計費。

🧮 北北基跳表

起跳 85、續程、延滯、夜間、春節加成，費率可在設定調整。

🛣️ 高速自動偵測

離線國道圖資確認 GPS 真的在高速公路上才亮燈、估過路費，可手動覆寫。

🖥️ 全螢幕營業模式

綠底跳表填滿整個螢幕、橫向呈現，像真機。

🔋 螢幕常亮

行車中 Wake Lock 防熄屏。

📊 今日統計

趟次 / 收入累計，存於手機本機。

使用方式

全螢幕營業模式

  1. 點主畫面上方 ⤢ 全螢幕營業模式
  2. 把手機橫著架好（直握會自動轉 90° 填滿螢幕）
  3. 按 計程計時 開始營業
  4. 右上角 ✕ 退出

加到主畫面（變 App）

用手機瀏覽器選單選「加到主畫面 / 安裝」，即可像原生 App 開啟、離線使用。

GPS 訊號判讀

  • 定位中…（閃爍）：尚未取得定位
  • ▮▮▮ 8m：已定位，亮格越多＝越準；數字是精度（公尺，越小越好）
  • 個位數～十幾公尺很好；> 80m 表示訊號差（隧道 / 高樓）會變黃 / 紅
  • 時速 XX：目前即時車速；另有「均速」為整趟平均

收費規則（北北基 2023-04-01 起）

項目規則
起跳1.25 公里內 85 元
續程每 200 公尺加 5 元
延滯計時車速 < 5 km/h 時，每 60 秒加 5 元（與續程不雙重計算）
夜間加成23:00–06:00，每趟 +20（依上車時間判定）
春節加成每趟 +30（手動開啟）
過路費國道小客車約 1.2 元/公里，實報實付由乘客負擔

高速公路自動偵測（離線國道圖資）

自動模式下，用內建的離線國道圖資比對 GPS 位置，確認真的在高速公路上才亮藍燈、累計 ETC 過路費（國道線形約 1.7 萬點，首次載入後離線可用，不需連網查地圖）。圖資尚未載入時退回速度啟發式（持續 ≥ 80 km/h）。可用「高速公路」鈕手動覆寫（自動·圖資 → 手動開 → 手動關）。依北北基費率，上高速跳表照跳不變，差別只在過路費。

營業區域與費率設定

設定面板（右上 ⚙）最上方可選營業區域（大台北、桃園、新竹、台中、台南、高雄、宜蘭、花蓮…等 15 區），預設大台北；選了會自動帶入該區費率。也可逐項調整起跳、續程、延滯、夜間（固定額或%）、春節、過路費，自動記在手機本機，下次開啟沿用。各縣市為參考值，可自行微調；附「還原所選區域預設費率」。

每台手機有獨立識別碼與行程紀錄（本機保存）；設定中可「清除本機所有紀錄」。

技術 / 部署

Vite + TypeScript（strict）+ vite-plugin-pwa，無後端、無資料庫，全程瀏覽器端運算。靜態站以 Docker（nginx）部署於 Zeabur。
